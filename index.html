Welcome to Charleston

Charleston is a framework to build static HTML/CSS/JavaScript sites with any combination of HTML, HAML, CSS, SASS, JavaScript, and CoffeeScript, that's easy to put together and maintain.

Let's get started!

  1. I'm assuming you've run charleston my_project and are now looking at a sample page.
  2. Edit pages/index.haml, and create new HTML or HAML files in the pages directory to make new pages.
  3. Edit stylesheets/screen.sass, and create new CSS or SASS files in the stylesheets directory to make new stylesheets.
  4. Edit and create JavaScript or CoffeeScript files in the javascripts directory to make new script files.
  5. Add images to the images directory if you want.
  6. When you want to build the site, run rake and it will be created in the output directory.

Where to Put Files

Class Format Inputs Outputs
Page HTML pages/*.html output/*.html
Page HAML pages/*.haml output/*.html
Stylesheet CSS stylesheets/*.css output/stylesheets/*.css
Stylesheet SASS stylesheets/*.sass output/stylesheets/*.css
Script JavaScript javascripts/*.js output/javascripts/*.js
Script CoffeeScript javascripts/*.coffee output/javascripts/*.js
Image Any format images/* output/images/*

What Not To Do

  • Don't make any direct edits to the "output" directory. These files get clobbered when you update the source files.
